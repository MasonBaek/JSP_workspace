Cascading Style Sheets를 의미 
	선택자 {속성:속성값;       선택자  = CSS로 꾸밀 주인공
		속성: 속성값;
		속성: 속성값;
		,,,
		}

선택자 대표 종류 3가지

  • tag선택자
    - tag가 같으면 같은 스타일로 적용되므로 자동, 강제임
  • ID선택자
    - tag뒤에 id를 추가하고 #id명 형식으로 불러와서 적용.
    id가 고유한값이므로 한번만 적용가능하고 주로 구조적인 용도에씀.
  • Class선택자
    -tag뒤에class를 추가하고 .class명 형식으로 불러와서 적용.
    class는 중복적용이 가능하므로 동시에 같은 스타일을 적용할 수 있고
    디자인적인 용도에 많이 씀.
    (tag하나에 여러 class명을 적용할 수 있어서 스타일 조합이 가능)

CSS

  • External CSS - CSS파일을 독립적으로 저장하여 웹문서와 연결, 적용하는방식 통일성 있는 기획이 가능하며 유지보수도 용이
    <head> <link href="*css" rel="stylesheet"> </head>
  • Internal CSS - 웹문서의 head안에 CSS소스를 적용하는 방식 해당웹문서에만 적용됨
  • Inline CSS - 웹문서의 tag에 CSS소스를 적요하는 방식 소스가 길어지고 통일성 있는 기획이 어렵고 유지보수도 곤란하나 CSS적용 우선순위가 높음. body안의 tag만 수정가능한 경우 사용.
