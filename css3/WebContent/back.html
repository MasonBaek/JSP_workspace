background 배경

  • background-color:색상; 배경색상
  • background-image:url("이미지경로");배경이미지
    • background-repeat:반복 지정
      1. repeat 기본값. 양사방 반복
      2. repeat-x 가로로만 반복
      3. repeat-y 세로로만 반복
      4. no-repeat 반복 안함
    • background-position: ; 위치 지정
      1. left/center/right top/center/bottom 수평 수직순 조합
      2. npx npx / n% n% / nem nem / ... 수평 수직순 지정
    • background-size: ; 크기 지정
      1. cover 영역을 그림 하나로 전체 다덮기 (그림이 잘려나갈수도..)
      2. contain 그림 하나를 영역에 맞춰 덮기 (그림이 반복될 수 있음, 잘려나가지는 않음)
      3. npx npx / n% n% / nem nem / ... 수평 수직순 지정
    • background-attachment: ;스크롤 여부 지정
      1. scroll 기본값. 배경이미지 스크롤
      2. fixed 배경이미지
